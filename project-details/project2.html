Portfolio Project - Overview

Introduction

This portfolio project is a full-stack web application designed to showcase my projects, enable users to connect with me via social media, and allow them to submit messages directly to me.

Frontend Development

The frontend of this portfolio is developed using:

  • HTML, CSS, and JavaScript: To create a responsive and interactive user interface.
  • Netlify Hosting: The frontend is deployed on GitHub and managed on Netlify for seamless and fast performance.
  • Animations and Styling: CSS animations and JavaScript were utilized to enhance the user experience.

Backend Development

The backend of this project is built using:

  • Node.js and Express.js: To handle server-side logic and API requests.
  • MongoDB Atlas: A cloud-based database used to securely store user messages.
  • Heroku Hosting: The backend is hosted on Heroku to manage API requests and database connections.
  • REST API: Created API endpoints to handle form submissions and data retrieval.

Key Features

  • Showcases my projects with descriptions and links.
  • Includes links to my social media: LinkedIn, GitHub, Instagram, LeetCode, and Gmail.
  • Users can submit messages via a contact form, which are stored in MongoDB Atlas.
  • Ensures secure data handling and storage with form validation.
  • Fully responsive design, optimized for mobile, tablet, and desktop views.

Deployment Details

  • Frontend: Deployed on Netlify.
  • Backend: Hosted on Heroku.
  • Database: Stored on MongoDB Atlas.

Future Improvements

Planned updates and enhancements include:

  • Adding an admin panel for managing messages.
  • Implementing authentication for restricted content.
  • Enhancing the UI/UX with improved animations.
